#  Promenades en Catamaran

Application web de gestion de promenades en catamaran, réalisée dans le cadre du BTS SIO.

##  Description

Site web permettant de gérer et réserver des promenades en catamaran. Il propose deux types de catamarans : loisir et sportif, avec un système de réservation, d'authentification et un espace administrateur.

##  Fonctionnalités

- Inscription / Connexion des utilisateurs
- Réservation de promenades en catamaran
- Espace personnel (mes réservations, mon profil)
- Espace administrateur (dashboard, gestion des réservations)
- Pages détaillées par destination (Atlantique, Calanque, Côte d'Azur, Golfe, Méditerranée, La Rochelle)
- Page événements, infos pratiques et contact

##  Technologies utilisées

- **Python** / **Flask** (backend)
- **MySQL** (base de données)
- **HTML / CSS** (frontend)
- **MAMP** (environnement de développement local)
